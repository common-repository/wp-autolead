WordPress Autolead plugin for survey and lead capture.

== Description ==

WordPress AutoLead - The most effective survey tool for wordpress, discover what your customers and website visitors really think.

<strong>Main Features:</strong>

1) WordPress Autolead allows you to create unlimited surveys and track the results. These surveys can be used for existing customers or new website visitors.

2) WordPress Autolead can be used as a lead capture system, asking your visitors a few questions and then prompting the user to enter their email address at the end of the survey.

<strong>So what is WordPress AutoLead and what can it do for you?</strong>

<ul>
	<li>Create unlimited surveys</li>
	<li>Complete design creativity</li>
	<li>2 step lead capture</li>
	<li>All Major autoresponders integrated (including Automail)</li>
	<li>Integrates with any wordpress theme</li>
	<li>Fully responsive forms</li>
	<li>Works with all browsers and devices</li>
	<li>drag and drop survey builder</li>
</ul>


== Installation ==

1. Upload the 'wp-autolead' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done!
4. Go to Dhasboard -> AutoLead.
